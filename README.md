# PHP CRUD Blog Application

A simple Blog Application built using PHP and MySQL with CRUD operations and user authentication.

## Features

- User Registration
- User Login
- Password Hashing
- Session Management
- Create Blog Posts
- Read/View Posts
- Update Posts
- Delete Posts

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- XAMPP
